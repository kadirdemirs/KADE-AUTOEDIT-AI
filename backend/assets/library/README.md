# KADE Asset Library

Put your own licensed Premiere assets in category folders here. KADE scans this
folder and shows the assets in the Library tab.

Suggested folder names:

- `transitions`
- `media-animations`
- `templates`
- `titles-lower-thirds`
- `social-media`
- `typography`
- `gaming-elements`
- `color-presets`
- `effects-overlays`
- `text-presets-mogrt`
- `logo-intros`
- `stories`
- `devices`
- `messages`

Supported files:

- `.mogrt`
- `.mp4`, `.mov`, `.m4v`, `.webm`
- `.png`, `.jpg`, `.jpeg`, `.webp`, `.gif`
- `.wav`, `.mp3`, `.aif`, `.aiff`, `.m4a`
- `.cube`, `.look`
- `.prfpset`
